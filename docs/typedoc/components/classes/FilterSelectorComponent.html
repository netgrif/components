Class FilterSelectorComponent

Allows user to choose a {@link Filter} from the {@link FilterRepository}.

Publishes events to the {@link SideMenuControl} object when:

  • filter is selected by the user. Message: New selected filter, Data: is either the selected filter or undefined if the user deselected the filter

  • filter selection is confirmed by the user. Message: Selected filter was confirmed, Data: the selected filter. If the user didn't select any filter this event will not be published.
