Interface TaskSearchRequestBody

Describes objects that are used to search/filter tasks from backend. Returned tasks must fulfill all provided criteria.

This object is used as part of a Filter object for [searchTask()]TaskResourceService#searchTask method in TaskResourceService.

interface TaskSearchRequestBody {

}
