Class CaseStringId

A utility class that exists for inheritance by simpler category instances.

The only configuration input is an [OPERATOR]SearchInputType#OPERATOR and a FormControl instance is available for it as well.

For [AutocompleteCategories]AutocompleteCategory use NoConfigurationAutocompleteCategory instead.
