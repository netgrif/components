Class CategoryFactory

Can be used to generate Category class instances.

Can only generate instances of Categories that take OperatorService as their first argument, LoggerService as their second argument and OptionalDependencies object is passed as the third. null is passed as the third argument otherwise. You can extend this class to support your Categories, but make sure that an injection token for this classes name is still provided for the library components that use it.
